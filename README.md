# MS Sustentável - Plataforma de Certificação RTRS

Bem-vindo ao repositório central do **Programa MS Sustentável**. Esta plataforma foi desenvolvida para digitalizar e auditar todo o processo de adequação e certificação de propriedades rurais no estado do Mato Grosso do Sul, sob os rigorosos critérios da RTRS (Round Table on Responsible Soy).

## 📚 Documentação e Contexto da Arquitetura

O projeto deixou a fase de testes protótipo e possui uma fundação arquitetural e de negócios **blindada e definitiva**. Todo o conhecimento necessário para novos desenvolvedores, engenheiros de devOps ou gestores do projeto foi consolidado na pasta `contexto/`.
